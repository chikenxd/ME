# Flash Powder Factory AFK’er

Simple script that travels to Flash Powder Factory, joins a game, and AFKs while automatically rejoining.

## Requirements

- Rogues’ Den unlocked
- Lodestones available
- No gear required

## Features

- Auto travel to Rogues’ Den
- Quick start via Brian O’Richard
- Auto dialogue handling
- AFKs inside the arena
- Rejoins rounds automatically
- Basic anti-idle
- Simple GUI with runtime + Thaler tracking

## Setup

1. Start the script anywhere  
2. Click **Start**  
3. Done  

## Notes

- Works best on Spotlight
- Empty inventory recommended (avoids banking step)
- Script runs indefinitely until stopped

Designed to be minimal, stable, and low-interaction.
